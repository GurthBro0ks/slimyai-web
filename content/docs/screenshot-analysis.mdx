## API Reference

### Upload & Analyze

```typescript
POST /api/screenshot
Content-Type: multipart/form-data

Parameters:
- screenshots: File[] (optional) - Image files to upload and analyze
- imageUrls: string[] (optional) - URLs of images to analyze
- userId: string (required) - User identifier
- type: ScreenshotType (optional) - Analysis type override
- analyze: boolean (optional) - Whether to perform AI analysis
- options: object (optional) - Advanced analysis options

Response:
{
  success: boolean,
  uploaded: number,
  processed: number,
  analyzed: number,
  results: ScreenshotAnalysisResult[],
  summary: {
    totalImages: number,
    validImages: number,
    invalidImages: number,
    analyzedImages: number,
    averageConfidence: number,
    screenshotType: string
  },
  warnings?: string[]
}
```

### Retrieve Analyses

```typescript
GET /api/screenshot

Query Parameters:
- userId: string (required) - User identifier
- type: ScreenshotType (optional) - Filter by analysis type
- limit: number (optional) - Results per page (default: 20)
- offset: number (optional) - Pagination offset (default: 0)
- search: string (optional) - Search term for title/description/tags
- tags: string[] (optional) - Filter by tags (comma-separated)

Response:
{
  success: boolean,
  results: ScreenshotAnalysisResult[],
  pagination: {
    total: number,
    limit: number,
    offset: number,
    hasMore: boolean
  },
  filters: {
    userId: string,
    type?: string,
    search?: string,
    tags?: string[]
  }
}
```

### Analysis Result Structure

```typescript
interface ScreenshotAnalysisResult {
  id: string;
  screenshotType: ScreenshotType;
  timestamp: Date;
  imageUrl: string;
  analysis: {
    title: string;
    description: string;
    summary: string;
    data: Record<string, any>;
    insights: string[];
    recommendations: string[];
    tags: string[];
  };
  metadata: {
    confidence: number;        // 0-1 quality score
    processingTime: number;   // milliseconds
    modelUsed: string;        // AI model identifier
    extractedText?: string;   // OCR text extraction
    dominantColors?: string[]; // Color palette
    detectedElements?: string[]; // UI elements found
  };
  rawResponse?: any; // Raw AI response data
}
```

## Troubleshooting

### Upload Issues

- **File Size Limits**: Ensure images are under size restrictions
- **Format Support**: Use supported image formats (PNG, JPG, WebP)
- **Network Issues**: Check internet connection and retry uploads
- **Permissions**: Verify file access and upload permissions

### Analysis Issues

- **Low Confidence Scores**: Try higher quality or clearer screenshots
- **Incomplete Results**: Ensure screenshots show complete data
- **Type Mismatch**: Select correct analysis type or use "custom"
- **Processing Timeouts**: Large images may take longer to process

### Comparison Problems

- **Different Types**: Compare screenshots of similar content types
- **Inconsistent Data**: Ensure comparable metrics between screenshots
- **Quality Variations**: Use screenshots of similar quality for best results
